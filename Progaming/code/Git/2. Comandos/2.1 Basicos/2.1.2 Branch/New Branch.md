### Crea una nueva rama:

~~~
git branch <nombre-de-la-rama>
~~~

### Ejemplo:

~~~
git branch testing
~~~

### Crear y moverse a la nueva rama:
~~~
git checkout -b <nombre-de-tu-rama>
~~~
