### Enviar cambios al servidor remoto:
~~~
git push <nombre-remoto> <nombre-de-tu-rama>
~~~
### Aviso:
- **Git push** solamente carga los cambios que han sido confirmados, por lo que es necesaio guardar los cambios usando un **commit**
